Pig Latin Translator:

get user input and display the translated phrase when the button is clicked. Features user input, animations, hover events, mouse move events.

Jquery Puzzle:

Use jquery to create puzzle pieces and drop zones. Match the piece to the correct drop zone. Match all pieces to win. Choose a harder difficulty to get more pieces.

Light Switch:

Use function oriented objects to create separate rooms that have their own 'state'. Click the lightswitch to light up the room. Click the plus or minus buttons to add and remove rooms.
